## Legend

| Status | Meaning |
|---|---|
| `backlog` | Ticket created, not started |
| `in_progress` | A developer agent is building it |
| `built` | Implementation done, waiting for verification |
| `qa` | QA / security / audit agents are reviewing |
| `debugging` | Findings returned, debugger agent is fixing |
| `done` | Verified and accepted |
| `blocked` | Cannot proceed — see ticket notes |
